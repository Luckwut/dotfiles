# `niri-ocr.sh`
Install the following packages
- [grim](https://archlinux.org/packages/extra/x86_64/grim/)
- [imagemagick](https://archlinux.org/packages/extra/x86_64/imagemagick/)
- [slurp](https://archlinux.org/packages/extra/x86_64/slurp/)
- [tesseract](https://archlinux.org/packages/extra/x86_64/tesseract/)
    - And language data [tesseract-data-eng](https://archlinux.org/packages/extra/any/tesseract-data-eng/) + [tesseract-data-ind](https://archlinux.org/packages/extra/any/tesseract-data-ind/)
